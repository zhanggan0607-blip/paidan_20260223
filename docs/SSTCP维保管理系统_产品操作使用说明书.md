## 1. 系统概述

### 1.1 系统简介

SSTCP维保管理系统是一套面向运维维保行业的综合管理平台，用于管理维保项目的全生命周期，包括项目信息管理、工单流转、巡检维修、零星用工、备品备件、维修工具、维保日志和部门周报等核心业务。

### 1.2 系统架构

系统采用前后端分离架构，包含以下三个子系统：

| 子系统 | 技术栈 | 访问方式 | 适用场景 |
|--------|--------|----------|----------|
| PC管理端 | Vue 3 + Element Plus | 浏览器访问 `https://paidan.sstcp.top` | 管理人员日常办公 |
| H5移动端 | Vue 3 + Vant UI | 手机浏览器访问 `https://paidan.sstcp.top/h5/` | 运维人员现场操作 |
| 后端服务 | FastAPI + PostgreSQL | API接口 `/api/v1/` | 数据处理与存储 |

### 1.3 功能总览

```
SSTCP维保管理系统
├── 统计分析 — 数据看板、工单统计
├── 维保项目管理 — 项目信息、维保计划、超期/临期提醒
├── 工单管理 — 定期巡检、临时维修、零星用工
├── 备品备件管理 — 领用、归还、库存、入库
├── 维修工具管理 — 领用、归还、库存、入库
├── 维保日志与周报 — 日志填报、日志查询、周报填报、周报查询
└── 系统管理 — 人员管理、客户管理、巡检事项管理
```

## 2. 角色与权限

### 2.1 角色定义

系统共有4种角色，权限由高到低排列如下：

| 角色 | 角色代码 | 说明 |
|------|----------|------|
| 管理员 | admin | 系统最高权限，可管理所有功能和数据 |
| 部门经理 | department_manager | 管理本部门项目、审批工单、查看周报 |
| 材料员 | material_manager | 管理备品备件和维修工具的库存与入库 |
| 运维人员 | employee | 现场操作人员，填报日志、领用物资 |

## 3. PC端操作指南

### 3.1 登录与密码管理

#### 3.1.1 登录系统

**操作步骤**:

1. 打开浏览器（推荐Chrome/Edge），访问系统地址 `https://paidan.sstcp.top`
2. 在登录页面输入用户名和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转至首页（项目信息管理页面）

**注意事项**:
- 首次登录或密码过期时，系统会强制跳转至修改密码页面
- 连续多次登录失败会触发频率限制，需等待一段时间后重试
- 登录Token有效期为24小时，过期后需重新登录

#### 3.1.2 修改密码

**操作步骤**:

1. 登录系统后，点击右上角用户头像
2. 选择「修改密码」选项
3. 输入当前密码和新密码（需输入两次确认）
4. 点击「确认修改」按钮

**密码要求**:
- 密码长度不少于6位
- 新密码不能与当前密码相同

#### 3.1.3 管理员重置密码

**操作步骤**（仅管理员/部门经理）:

1. 进入「系统管理 → 人员管理」
2. 找到目标人员，点击操作列的「重置密码」按钮
3. 确认重置操作
4. 重置后该人员下次登录需修改密码

### 3.2 统计分析

**访问路径**: 侧边栏 → 统计分析

**功能说明**: 提供维保业务的数据看板，直观展示年度工单统计、完成率、项目排名等关键指标。

**页面内容**:

| 统计项 | 说明 |
|--------|------|
| 年度工单总数 | 当前年度所有工单数量汇总 |
| 临期工单数 | 7天内即将到期的工单数量 |
| 超期工单数 | 已超过截止日期的工单数量 |
| 已完成工单数 | 状态为已完成的工单数量 |
| 准时完成率 | 按时完成的工单占比 |
| 年度TOP5项目 | 工单数量排名前5的项目 |
| 年度TOP5维修单 | 临时维修单数量排名 |
| 运维人员工单统计 | 各运维人员的工单完成情况 |

**操作说明**:
- 页面数据自动加载，无需手动操作
- 统计数据基于当前年度实时计算
- 可点击各统计卡片查看明细数据

### 3.3 维保项目管理

#### 3.3.1 项目信息管理

**访问路径**: 侧边栏 → 维保项目管理 → 项目信息管理

**功能说明**: 管理维保项目的基础信息，包括项目编号、名称、客户、地址、合同日期等。

**操作步骤 — 新增项目**:

1. 点击「新增项目」按钮
2. 填写项目信息：
   - **项目编号**: 系统自动生成，格式如 `TQ2023423`
   - **项目名称**: 必填，输入项目名称
   - **客户名称**: 选择关联客户
   - **项目地址**: 输入项目所在地址
   - **合同开始日期/结束日期**: 选择合同有效期
   - **项目描述**: 填写项目简介
3. 点击「保存」按钮

**操作步骤 — 编辑项目**:

1. 在项目列表中找到目标项目
2. 点击操作列的「编辑」按钮
3. 修改需要更新的字段
4. 点击「保存」按钮

**操作步骤 — 删除项目**:

1. 在项目列表中找到目标项目
2. 点击操作列的「删除」按钮
3. 确认删除操作

> **注意**: 删除项目会级联删除该项目的维保计划、工单等关联数据，请谨慎操作。

**筛选与搜索**:
- 支持按项目名称模糊搜索
- 支持按客户名称筛选
- 列表支持分页浏览

#### 3.3.2 维保计划管理

**访问路径**: 侧边栏 → 维保项目管理 → 维保计划管理

**功能说明**: 制定和管理维保计划，关联项目与巡检周期。

**操作步骤 — 新增维保计划**:

1. 点击「新增计划」按钮
2. 填写计划信息：
   - **计划编号**: 系统自动生成
   - **关联项目**: 选择所属项目
   - **计划类型**: 选择维保类型
   - **执行周期**: 设置巡检频率
   - **计划开始/结束日期**: 设定计划有效期
   - **执行状态**: 待执行/执行中/已完成
   - **完成率**: 设置当前完成百分比
3. 点击「保存」按钮

**操作步骤 — 更新执行状态**:

1. 找到目标计划，点击「更新状态」
2. 选择新的执行状态
3. 可同步更新完成率
4. 点击「确认」按钮

**筛选功能**:
- 按项目名称筛选
- 按执行状态筛选
- 按日期范围筛选
- 查看即将到期的计划

#### 3.3.3 项目超期提醒

**访问路径**: 侧边栏 → 维保项目管理 → 项目超期提醒

**功能说明**: 展示已超过合同截止日期或计划截止日期的项目，便于及时处理。

**操作说明**:
- 列表自动展示所有超期项目
- 支持按项目名称搜索
- 显示超期天数，便于优先处理

#### 3.3.4 项目临期提醒

**访问路径**: 侧边栏 → 维保项目管理 → 项目临期提醒

**功能说明**: 展示7天内即将到期的项目，提前预警。

**操作说明**:
- 列表自动展示所有临期项目
- 显示剩余天数
- 支持按项目名称搜索

### 3.4 工单管理

#### 3.4.1 定期巡检单查询

**访问路径**: 侧边栏 → 工单管理 → 定期巡检单查询

**功能说明**: 查看和管理定期巡检工单，包括巡检记录的查看、审批等操作。

**工单状态流转**:

```
草稿 → 已提交 → 已审批（通过/退回）
  ↑       ↓
  └── 撤回 ─┘
```

**操作步骤 — 查看巡检单**:

1. 在列表中浏览所有巡检工单
2. 点击工单编号或「查看」按钮进入详情页
3. 详情页展示：基本信息、巡检事项、巡检记录、现场照片、操作日志

**操作步骤 — 审批巡检单**（管理员/部门经理）:

1. 在列表或详情页中找到待审批的工单
2. 点击「审批」按钮
3. 选择「通过」或「退回」
4. 如退回，需填写退回原因
5. 点击「确认」按钮

**筛选功能**:
- 按项目名称筛选
- 按工单状态筛选（草稿/已提交/已审批/已退回）
- 按日期范围筛选
- 按创建人筛选

#### 3.4.2 临时维修单查询

**访问路径**: 侧边栏 → 工单管理 → 临时维修单查询

**功能说明**: 查看和管理临时维修工单。

**操作步骤 — 查看维修单**:

1. 在列表中浏览所有临时维修工单
2. 点击工单编号进入详情页
3. 详情页展示：维修信息、现场照片、维修结果、操作日志

**操作步骤 — 审批维修单**（管理员/部门经理）:

1. 找到待审批的工单
2. 点击「审批」按钮
3. 选择通过或退回
4. 确认操作

**导出PDF**:
- 在维修单详情页，点击「导出PDF」按钮
- 系统自动生成PDF文件并下载

#### 3.4.3 零星用工单查询

**访问路径**: 侧边栏 → 工单管理 → 零星用工单查询

**功能说明**: 查看和管理零星用工工单，包括施工人员信息。

**操作步骤 — 查看用工单**:

1. 在列表中浏览所有零星用工工单
2. 点击工单编号进入详情页
3. 详情页展示：用工信息、施工人员列表（含身份证照片）、现场照片、操作日志

**操作步骤 — 审批用工单**（管理员/部门经理）:

1. 找到待审批的工单
2. 点击「审批」按钮
3. 选择通过或退回
4. 确认操作

**导出PDF**:
- 在用工单详情页，点击「导出PDF」按钮
- 系统自动生成包含施工人员信息的PDF文件

### 3.5 备品备件管理

#### 3.5.1 备品备件领用

**访问路径**: 侧边栏 → 备品备件管理 → 备品备件领用

**功能说明**: 登记备品备件的领用记录，领用时自动扣减库存。

**操作步骤 — 领用备品备件**:

1. 点击「新增领用」按钮
2. 填写领用信息：
   - **备品备件**: 从库存列表中选择
   - **领用数量**: 输入领用数量（不能超过库存数量）
   - **领用人**: 选择领用人
   - **关联项目**: 选择所属项目
   - **领用日期**: 选择领用日期
   - **备注**: 填写用途说明
3. 点击「确认领用」按钮
4. 系统自动扣减对应库存

**注意事项**:
- 领用数量不能超过当前库存数量
- 领用后库存实时更新

#### 3.5.2 备品备件归还

**访问路径**: 侧边栏 → 备品备件管理 → 备品备件归还

**功能说明**: 登记备品备件的归还记录，归还时自动回补库存。

**操作步骤 — 归还备品备件**:

1. 在领用记录列表中找到需要归还的记录
2. 点击「归还」按钮
3. 输入归还数量
4. 点击「确认归还」按钮
5. 系统自动回补对应库存

#### 3.5.3 备品备件库存

**访问路径**: 侧边栏 → 备品备件管理 → 备品备件库存

**功能说明**: 查看备品备件库存列表，管理入库记录。

1. 点击「新增入库」按钮
2. 填写入库信息：
   - **备品备件名称**: 输入名称
   - **规格型号**: 输入规格
   - **单位**: 选择计量单位
   - **入库数量**: 输入数量
   - **供应商**: 输入供应商
   - **入库日期**: 选择日期
3. 点击「确认入库」按钮

**操作说明**:
- 库存列表展示所有备品备件的当前库存数量
- 支持按名称搜索
- 可查看入库历史记录

### 3.7 维保日志与周报

**访问路径**: 侧边栏 → 维保日志 → 新报维保日志

**功能说明**: 填报每日维保日志，记录当天工作内容。

2. 填写日志信息：

## 5. 常见问题与故障排除

### 5.1 登录相关

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 登录提示"用户名或密码错误" | 输入信息有误 | 检查用户名和密码，注意大小写 |
| 登录提示"操作过于频繁" | 短时间内多次登录失败 | 等待1-2分钟后重试 |
| 登录后自动跳转到修改密码页 | 首次登录或密码被重置 | 按要求修改密码后即可正常使用 |
| Token过期被踢出 | 登录超过24小时 | 重新登录即可 |

### 5.2 工单相关

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 无法提交工单 | 必填字段未填写 | 检查所有必填字段是否已填写 |
| 无法撤回工单 | 工单已被审批 | 已审批的工单无法撤回，联系管理员处理 |
| 工单列表为空 | 权限限制或筛选条件 | 普通用户只能看到自己的工单，检查筛选条件 |

### 5.3 OCR识别相关

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| OCR识别失败 | 网络不稳定或图片模糊 | 确保网络畅通，重新拍照保持清晰 |
| 提示"操作过于频繁" | OCR调用频率超限 | 等待1分钟后重试，或手动输入信息 |
| 身份证信息识别不全 | 拍照角度或光线问题 | 正面拍摄，确保光线充足，身份证完整入框 |

### 5.4 图片上传相关

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 图片上传失败 | 网络不稳定或图片过大 | 切换网络后重试，系统会自动压缩图片 |
| 图片显示"加载失败" | 网络问题或Token过期 | 刷新页面或重新登录 |
| 拍照后图片不清晰 | 手机摄像头对焦问题 | 点击屏幕对焦后再拍摄 |

### 5.5 备品备件/工具相关

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 领用提示库存不足 | 库存数量不够 | 联系材料员补充库存 |
| 无法入库 | 权限不足 | 入库操作需要管理员/部门经理/材料员权限 |
| 归还数量超过领用数量 | 输入错误 | 归还数量不能超过领用数量 |
